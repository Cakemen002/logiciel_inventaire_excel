# logiciel_inventaire_excel

Ce fichier excel permet de gérer l'inventaire d'une entreprise. 
Toute transaction est possible avec ce ligiciel. De la soumission vers la facturation et de l'achat jusqu'à la réception. 
Ce logiciel vous permet entre autre de générer des rapports de facturation qui vous permet de bien visualiser vos transaction, et ce, trié de manière que vous le souhaitez.
